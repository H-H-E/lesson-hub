# Inquiry Skills

## What is Inquiry?

Learning by asking questions and finding answers.

## Skills

### Asking Questions
- Wonder about the world
- Be curious
- Question everything

### Investigating
- Plan tests
- Collect data
- Observe carefully

### Explaining
- Share findings
- Connect to ideas
- Reflect on learning

---

## Questions to Ask
- What do I notice?
- What do I wonder?
- How can I find out?
- What does this mean?
