# K-12 STEM Story Arc
## The Journey: "The Explorer's Handbook"

### Core Metaphor
All of STEM education is framed as preparing students to become **Explorers** of the natural world. Each grade level represents a new chapter in their journey as they discover, understand, and eventually contribute to the world.

---

## Cross-Cutting Creative Elements

### The Explorer's Journal (All Grades)
A through-portfolio that travels with students, with:
- "Discovery entries" (observations)
- "Wonder questions" (curiosity)
- "Aha moments" (insights)
- "Connection maps" (how topics link)

### The Classroom Ecosystem
Each class builds:
- A class mascot (connects to biology)
- A "question board" (inquiry)
- A "design lab" space (engineering)
- A "nature corner" (ongoing observation)

### Seasonal Stories
Each unit connects to the seasons:
- **Fall:** Harvest/collection (data)
- **Winter:** Reflection/reading (theory)
- **Spring:** Growth/experimentation (practice)
- **Summer:** Celebration/sharing (presentation)

## Creativity Integration Points

### Every Lesson Includes:

1. **The Hook** - A mystery or story question
2. **The Discovery** - Hands-on exploration
3. **The Connection** - Link to the larger story
4. **The Create** - Student expression of learning

### Metaphors That Bind:

| Concept | Metaphor |
|---------|----------|
| Cells | Building blocks/team members |
| Energy | Currency/power |
| Ecosystems | Family/community |
| Forces | Language of motion |
| Chemistry | Cooking/transformations |
| Physics | Game rules |
| Technology | Tools in our toolkit |

---

## Implementation Guide

### For Each Grade Level:

1. **Theme Song** - Simple class chant about the topic
2. **Story Character** - A mascot that "learns alongside" students
3. **Quest** - The main project/challenge of the unit
4. **Badge** - Recognition of mastery
5. **Legacy** - How this learning helps others

### Sample: Grade 1 "Living Neighbors"

- **Theme Song:** "We're All Neighbors" (to tune of "If You're Happy")
- **Character:** "Scout the Squirrel" (seasonal visitor)
- **Quest:** Create a field guide for our schoolyard
- **Badge:** "Neighbor Explorer" pin
- **Legacy:** Share guide with kindergarten
