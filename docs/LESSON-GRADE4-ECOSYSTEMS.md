# Grade 4 Science: Ecosystems
## Systems Literacy Unit | Lesson 1 of 5

---

## 1. SMART Objectives

By the end of this lesson, students will be able to:

- **Specific:** Define the term "ecosystem" and identify at least 3 components (producers, consumers, decomposers) within a given ecosystem
- **Measurable:** Correctly categorize 5 different organisms as producers, consumers, or decomposers with 80% accuracy
- **Achievable:** Explain using their own words how energy flows through a simple food chain
- **Relevant:** Connect the concept to their local environment by identifying at least 2 ecosystems in their community
- **Time-bound:** Complete all activities within the 60-minute lesson period

**Systems Literacy Objective:** Students will begin to see themselves as interconnected participants within larger systems, recognizing that their choices affect ecosystem health.

## 3. Hook

### "The Invisible Web" (8 minutes)

**Step 1 - Mystery Box (3 min):**
Hold up a sealed box. Say: "Inside this box is something that keeps you alive right now. You can't see it, but you couldn't survive 5 minutes without it."

Invite 2-3 students to briefly smell the box (contains fresh soil, leaves, grass - earthy scents). Ask: "What do ecosystems have to do with you surviving?"

**Step 2 - The Connection Chain (5 min):**
Stand in the center of the room. Say: "You are not a lone creature wandering through a world of things. You are a node in a web."

Ring the chime. One student stands. Ask: "Who do you need to survive?" (Family, food, water)

Ring again - that student picks a partner. Continue: "Now who do THEY need?" Keep building until 8-10 students are connected by holding hands/arms.

**Key Question:** "What happens if we remove ONE person from this chain? What if we remove one SPECIES from an ecosystem?"

> **Teacher Note:** This kinesthetic activity creates embodied understanding of interconnection. Some students may be surprised that humans are part of "nature."

## 4. Lesson Sequence with Timing

| Time | Activity | Description |
|------|----------|-------------|
| **0:00-0:08** | Hook | "The Invisible Web" connection activity |
| **0:08-0:18** | Direct Instruction | Interactive vocabulary: ecosystem, producer, consumer, decomposer |
| **0:18-0:30** | Guided Practice | Sorting cards into categories with partner discussion |
| **0:30-0:42** | Independent Practice | Build-a-food-chain worksheet + journal entry |
| **0:42-0:52** | Technology Integration | p5.js ecosystem simulation exploration |
| **0:52-0:60** | Closure | Exit ticket + reflection |

### Detailed Sequence

**0:08-0:18 - Direct Instruction (10 min)**
- Display 4 quadrants on board: Forest, Pond, Desert, Your Backyard
- Co-create definitions using student-friendly language:
  - **Ecosystem:** A community where living things and non-living things interact
  - **Producer:** Makes its own food (plants, algae)
  - **Consumer:** Eats other living things
  - **Decomposer:** Breaks down dead things into nutrients
- Use think-pair-share: "Can you be more than one?"

**0:18-0:30 - Guided Practice (12 min)**
- Distribute shuffled ecosystem cards (organisms + elements: sun, water, soil, wolf, rabbit, grass, mushroom, hawk, fish, frog, bear, bee, flower, bacteria, dead leaf)
- Partners sort into 4 categories on desk
- Gallery walk to compare answers
- Whole-class discussion: "Was this easy or hard? Why?"

**0:30-0:42 - Independent Practice (12 min)**
- Students complete "Build Your Own Ecosystem" worksheet
- Draw a simple ecosystem with at least 3 food chains
- Write 2-3 sentences explaining energy flow
- Journal prompt: "What would happen if all the decomposers disappeared?"

**0:42-0:52 - Technology Integration (10 min)**
- Demonstrate p5.js simulation (Section 14)
- Students explore in pairs: "What happens when you remove one species?"
- Record observations in journal

**0:52-0:60 - Closure (8 min)**
- Exit ticket (3-2-1 format)
- 3 things you learned, 2 questions you still have, 1 connection to your life
- Volunteer shares
- Preview next lesson: "Tomorrow we'll explore food webs - ecosystems get even more connected!"

## 5. Three-Level Differentiation

### Level 1: Support (Students needing scaffolding)

**Instructional Adjustments:**
- Pre-teach vocabulary with picture cards before lesson
- Provide word banks on worksheets
- Seat near teacher for frequent check-ins
- Reduce number of items to sort (8 instead of 15)
- Allow verbal responses in addition to written

**Sample Modification:**
- Worksheet includes a partially completed food chain: Sun → Grass → ___ → Hawk
- Student fills in the blank and adds one more organism

**Assessment Flexibility:**
- Accept oral exit ticket
- Provide sentence stems: "An ecosystem is ___."

### Level 2: On-Level (Standard instruction)

**Standard Experience:**
- Full participation in all activities
- Complete all worksheets independently
- Standard assessment expectations

### Level 3: Extension (Students ready for deeper learning)

**Challenge Activities:**
- Research question: "How do invasive species affect ecosystem balance?"
- Create a "Super Consumer" profile: Design an organism that eats producers AND consumers - is this realistic? Why/why not?
- Advanced journal prompt: "If humans are part of ecosystems, what ethical responsibilities do we have?"

**Extension Project Option:**
- "Ecosystem in a Bottle" - design and maintain a closed ecosystem (2-week project)

**Acceleration:**
- Introduce food web concepts early
- Partner with Level 1 students as "ecosystem expert" mentors

## 7. Systems Literacy Connection

### Core Message: "You are not a lone creature..."

> **"You are not a lone creature wandering through a world of things. You are a node in a web—a thread in a tapestry. When you pull one strand, the whole fabric shifts."**

This lesson plants the seed of systems literacy by helping students discover:

1. **Interconnection:** Every organism affects others (the "web" metaphor)
2. **Boundaries:** Ecosystems have edges, but they're permeable
3. **Flow:** Energy moves through systems (not created/destroyed, just transferred)
4. **Feedback:** When one part changes, the whole system responds
5. **Your Place:** Humans are inside ecosystems, not outside them

### Systems Thinking Routine
After the lesson, add to classroom anchor chart:

**The Systems Thinking Starter:**
- "How is ___ connected to ___?"
- "What happens if we change ___?"
- "Who depends on ___?"
- "What would happen if ___ disappeared?"

## 11. Teacher Notes

### Common Misconceptions to Address

1. **"Humans aren't part of ecosystems"** → Emphasize: We breathe, eat, drink from ecosystems constantly
2. **"Only big animals matter"** → Highlight decomposers and producers
3. **"Ecosystems are separate"** → Show how ecosystems connect (migration, water cycles)
4. **"Nature is always in balance"** → Introduce change and disruption as normal

### Classroom Management Tips

- **Card Sorting:** Establish signal for "time's up" (chime, hand raise)
- **Movement:** Clear pathways for gallery walk
- **Technology:** Have backup paper version of simulation if devices fail
- **Discussion:** Use "talk tokens" - each student gets 2 tokens, one per share

### Timing Adjustments

- If running long on hook: Shorten gallery walk
- If students need more time: Extend simulation (valuable exploration)
- If lesson ends early: Add enrichment discussion or journal writing

### Assessment Data Use

- Review exit tickets same day to plan tomorrow's reinforcement
- Note students who need vocabulary support for follow-up
- Identify misconceptions to address in Lesson 2

### Safety Considerations

- If using natural materials (leaves, sticks) for activities, check for allergies
- Outdoor exploration requires appropriate supervision ratios
- Remind students not to taste or eat any ecosystem materials

## 14. p5.js Simulation

### Interactive Ecosystem Model

This hands-on simulation allows students to observe what happens when species are added, removed, or modified within an ecosystem.

**Access:** [p5.js Web Editor](https://editor.p5js.org/)

**Code:**

```javascript
// Ecosystem Simulation - Grade 4 Science
// Add this to p5.js web editor

let producers = [];
let consumers = [];
let decomposers = [];
let energy = 100;
let days = 0;

function setup() {
  createCanvas(600, 400);
  textSize(14);
  
  // Initialize with some starting organisms
  for(let i = 0; i < 8; i++) {
    producers.push(new Producer(random(width), random(height)));
  }
  for(let i = 0; i < 4; i++) {
    consumers.push(new Consumer(random(width), random(height)));
  }
  decomposers.push(new Decomposer(width/2, height/2));
}

function draw() {
  background(220, 240, 220);
  
  // Draw ecosystem zones
  noStroke();
  fill(180, 230, 180);
  ellipse(width/2, height/2, 300, 200); // Grassland zone
  
  // Display stats
  fill(0);
  text("Day: " + days, 20, 25);
  text("Energy in system: " + energy, 20, 45);
  text("Producers: " + producers.length, 20, 65);
  text("Consumers: " + consumers.length, 20, 85);
  text("Decomposers: " + decomposers.length, 20, 105);
  
  // Instructions
  text("Click to add PRODUCER (plants)", 400, 25);
  text("Press C to add CONSUMER", 400, 45);
  text("Press D to add DECOMPOSER", 400, 65);
  text("Press R to remove random species", 400, 85);
  text("Press SPACE to advance day", 400, 105);
  
  // Update and display organisms
  for(let p of producers) {
    p.update();
    p.display();
    p.grow();
  }
  
  for(let c of consumers) {
    c.update();
    c.display();
    c.eat(producers);
  }
  
  for(let dec of decomposers) {
    dec.update();
    dec.display();
  }
  
  // Check for ecosystem collapse
  if(producers.length === 0 && consumers.length > 0) {
    fill(255, 100, 100);
    textSize(24);
    text("ECOSYSTEM COLLAPSE!", width/2 - 100, height/2);
    textSize(14);
  }
}

function mousePressed() {
  producers.push(new Producer(mouseX, mouseY));
}

function keyPressed() {
  if (key === 'c' || key === 'C') {
    consumers.push(new Consumer(random(width), random(height)));
  }
  if (key === 'd' || key === 'D') {
    decomposers.push(new Decomposer(random(width), random(height)));
  }
  if (key === 'r' || key === 'R') {
    let r = random(3);
    if(r < 1 && producers.length > 0) producers.splice(floor(random(producers.length)), 1);
    else if(r < 2 && consumers.length > 0) consumers.splice(floor(random(consumers.length)), 1);
    else if(decomposers.length > 1) decomposers.splice(floor(random(decomposers.length)), 1);
  }
  if (key === ' ') {
    days++;
    // Daily processes
    for(let i = producers.length - 1; i >= 0; i--) {
      producers[i].energy -= 5;
      if(producers[i].energy <= 0) producers.splice(i, 1);
    }
    for(let i = consumers.length - 1; i >= 0; i--) {
      consumers[i].energy -= 10;
      if(consumers[i].energy <= 0) consumers.splice(i, 1);
    }
  }
}

// Organism classes
class Producer {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.size = 20;
    this.energy = 50;
    this.color = color(100, 200, 100);
  }
  
  update() {
    this.y += random(-0.5, 0.5);
    this.x += random(-0.5, 0.5);
  }
  
  grow() {
    if(random(100) < 5) {
      producers.push(new Producer(this.x + random(-20, 20), this.y + random(-20, 20)));
    }
  }
  
  display() {
    fill(this.color);
    noStroke();
    ellipse(this.x, this.y, this.size, this.size);
  }
}

class Consumer {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.size = 25;
    this.energy = 80;
    this.color = color(200, 100, 100);
  }
  
  update() {
    // Seek producers
    let nearest = null;
    let minDist = Infinity;
    for(let p of producers) {
      let d = dist(this.x, this.y, p.x, p.y);
      if(d < minDist) {
        minDist = d;
        nearest = p;
      }
    }
    if(nearest && minDist < 200) {
      this.x += (nearest.x - this.x) * 0.02;
      this.y += (nearest.y - this.y) * 0.02;
    } else {
      this.x += random(-2, 2);
      this.y += random(-2, 2);
    }
  }
  
  eat(producers) {
    for(let i = producers.length - 1; i >= 0; i--) {
      if(dist(this.x, this.y, producers[i].x, producers[i].y) < this.size/2 + producers[i].size/2) {
        producers.splice(i, 1);
        this.energy += 30;
        break;
      }
    }
  }
  
  display() {
    fill(this.color);
    noStroke();
    ellipse(this.x, this.y, this.size, this.size);
  }
}

class Decomposer {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.size = 15;
    this.color = color(150, 100, 50);
  }
  
  update() {
    this.x += random(-1, 1);
    this.y += random(-1, 1);
  }
  
  display() {
    fill(this.color);
    noStroke();
    rect(this.x - 8, this.y - 8, 16, 16);
  }
}
```

### Student Exploration Questions

3. **What is the "right balance" - can you create a stable ecosystem?**
4. **What happens if there are too many consumers?**
