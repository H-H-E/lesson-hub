# CSTA K-12 Computer Science Standards
## Aligned with Systems Literacy Vision

*Based on CSTA K-12 Computer Science Standards (Revised 2017)*

---

## Grade Bands

| Level | Grades | Description |
|-------|--------|-------------|
| 1A | K-2 | Early Foundation |
| 1B | 3-5 | Elementary |
| 2 | 6-8 | Middle School |
| 3A | 9-10 | High School Core |
| 3B | 9-12 | High School Specialty |

---

## Concepts

### AP - Algorithms and Programming
### CS - Computing Systems  
### DA - Data and Analysis
### IC - Impacts of Computing
### NI - Networks and the Internet

---

## K-2 (Level 1A) Standards

### Computing Systems
- 1A-CS-01: Select and operate appropriate tools to express ideas using digital representations
- 1A-CS-02: Use digital devices to communicate ideas with peers
- 1A-CS-03: Describe basic hardware and software

### Networks & Internet
- 1A-NI-04: Explain that devices connect to share information
- 1A-NI-05: Recognize that people use passwords to protect devices

### Data & Analysis
- 1A-DA-06: Collect and present data in visual formats
- 1A-DA-07: Identify patterns in data

### Algorithms & Programming
- 1A-AP-08: Model daily processes by creating algorithms
- 1A-AP-09: Create programs using sequencing
- 1A-AP-10: Decompose steps needed to solve problems
- 1A-AP-11: Create plans through collaborative design
- 1A-AP-12: Develop plans that describe sequence of events, goals, expected outcomes

### Impacts of Computing
- 1A-IC-13: Identify ways people use computing in daily life
- 1A-IC-14: Identify positive and negative impacts of computing

---

## 3-5 (Level 1B) Standards

### Computing Systems
- 1B-CS-01: Demonstrate how computer hardware and software work together
- 1B-CS-02: Describe internal and external components of devices
- 1B-CS-03: Troubleshoot device issues

### Networks & Internet
- 1B-NI-04: Model how data is transmitted across networks
- 1B-NI-05: Describe how the internet connects devices globally

### Data & Analysis
- 1B-DA-06: Collect, organize, represent data in cloud
- 1B-DA-07: Use data to highlight relationships
- 1B-DA-08: Identify patterns in data to make predictions

### Algorithms & Programming
- 1B-AP-08: Create programs using loops
- 1B-AP-09: Create programs using event detection
- 1B-AP-10: Decompose problems into smaller parts
- 1B-AP-11: Create programs using variables
- 1B-AP-12: Test and debug programs
- 1B-AP-13: Take on roles with teacher guidance
- 1B-AP-14: Create programs that include sequences, events, loops, conditionals

### Impacts of Computing
- 1B-IC-15: Discuss computing technologies that have changed the world
- 1B-IC-16: Brainstorm ways to improve accessibility of computing technologies

---

## 6-8 (Level 2) Standards

### Computing Systems
- 2-CS-01: Recommend improvements to design of computing devices
- 2-CS-02: Design projects that combine hardware and software
- 2-CS-03: Identify bugs in hardware and software

### Networks & Internet
- 2-NI-04: Model how data is routed across networks
- 2-NI-05: Explain how encryption works
- 2-NI-06: Discuss security issues

### Data & Analysis
- 2-DA-06: Describe stored data formats
- 2-DA-07: Represent data using multiple encoding schemes
- 2-DA-08: Collect data using computational tools
- 2-DA-09: Test computational models

### Algorithms & Programming
- 2-AP-10: Use flowcharts for algorithm design
- 2-AP-11: Create programs with nested loops
- 2-AP-12: Design programs with procedures
- 2-AP-13: Identify patterns in programs to generalize
- 2-AP-14: Create programs with conditional statements
- 2-AP-15: Discuss how programs manage user data

### Impacts of Computing
- 2-IC-16: Compare computing technologies' impacts across cultures
- 2-IC-17: Trade off ease of use and functionality
- 2-IC-18: Debug computational systems

---

## 9-10 (Level 3A) - High School Core

### Computing Systems
- 3A-CS-01: Explain how abstractions hide details in computing systems
- 3A-CS-02: Compare levels of abstraction in computing
- 3A-CS-03: Design hardware and software interfaces

### Networks & Internet
- 3A-NI-04: Describe the tradeoffs in how data moves through networks
- 3A-NI-05: Explain how encryption protects data
- 3A-NI-06: Discuss security policies

### Data & Analysis
- 3A-DA-06: Transform data to remove noise
- 3A-DA-07: Analyze data from multiple sources
- 3A-DA-08: Explain how models simulate real-world phenomena

### Algorithms & Programming
- 3A-AP-10: Use lists to simplify solutions
- 3A-AP-11: Evaluate algorithms for efficiency
- 3A-AP-12: Compare and contrast various sorting algorithms
- 3A-AP-13: Create programs using libraries
- 3A-AP-14: Develop plans describing program organization
- 3A-AP-15: Document programs for readability

### Impacts of Computing
- 3A-IC-16: Compare the beneficial and harmful effects of computing innovations
- 3A-IC-17: Discuss the ethics of sharing information digitally
- 3A-IC-18: Examine biases in algorithms
- 3A-IC-19: Understand consequences of online actions

---

## 9-12 (Level 3B) - High School Specialty

### Computing Systems
- 3B-CS-01: Categorize and explain hardware and software layers
- 3B-CS-02: Compare techniques for detecting software bugs
- 3B-CS-03: Compare ways software could be licensed

### Networks & Internet
- 3B-NI-04: Trace how data moves through the internet
- 3B-NI-05: Explain protocols used in network communication
- 3B-NI-06: Discuss security measures to protect information

### Data & Analysis
- 3B-DA-06: Create datasets using various collection methods
- 3B-DA-07: Create and visualize data models
- 3B-DA-08: Apply computational techniques to analyze data

### Algorithms & Programming
- 3B-AP-09: Compare search algorithms
- 3B-AP-10: Compare data structure tradeoffs
- 3B-AP-11: Design and implement algorithms with recursion
- 3B-AP-12: Evaluate algorithms using mathematical reasoning
- 3B-AP-13: Develop a program working in teams

### Impacts of Computing
- 3B-IC-14: Discuss computing innovations' impacts on global economy
- 3B-IC-15: Debate implications of computing technologies in justice
- 3B-IC-16: Propose ways to address digital divide
