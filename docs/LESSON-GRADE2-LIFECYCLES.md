# Grade 2 Science: Life Cycles
## A Systems Literacy Approach

**Grade Level:** 2  
**Subject:** Science  
**Duration:** 60 minutes (can be split across 2 sessions)  
**Theme:** Understanding Life as a System

---

## 1. SMART Objectives

By the end of this lesson, students will be able to:

1. **Identify** the four stages of a butterfly's life cycle (egg → caterpillar → chrysalis → butterfly) and label them correctly with 90% accuracy.

2. **Explain** how each stage of a life cycle depends on the previous stage, using the phrase "First, then, next, finally" to describe the sequence.

3. **Compare** the life cycle of a butterfly with at least one other animal (e.g., frog, chicken) by identifying similarities and differences.

4. **Design** a simple diagram showing how living things in a garden ecosystem are connected through their life cycles (systems thinking).

## 3. Hook (5-7 minutes)

### "The Mystery Box"

**Setup:** Bring in a sealed box with small air holes. Tell students there's something alive inside, but it's a secret!

**Script:**
> "Class, I have a mystery in this box. It's alive! I need some scientists to help me figure out what it could be. Let me give you some clues..."

**Clues:**
1. "It starts very, very small—so small you can barely see it."
2. "It changes its body completely—not once, but THREE times!"
3. "It has six legs but doesn't always have them."
4. "It loves to eat leaves, but one day it will fly through gardens."

**Engagement:** Have students whisper their guesses. After 3-4 guesses, reveal... it's a butterfly! Let a painted lady butterfly (or image) emerge gently from the box.

**Transition:** "Today we're going to become life cycle scientists. We'll discover how ALL living things grow and change—and how they're connected like a big team!"

## 4. Lesson Sequence

### Part 1: Direct Instruction (15 minutes)

**A. Introduce the Butterfly Life Cycle**
- Display large poster showing all 4 stages
- Use finger motions for each stage:
  - Egg: pinch fingers together (tiny!)
  - Caterpillar: wiggle fingers walking
  - Chrysalis: wrap arms around self
  - Butterfly: flutter hands

**B. Systems Connection Dialogue**
> "Think about a team. Every player has a job, right? Well, in a life cycle, EVERY stage has a job! The egg's job is to START the life. The caterpillar's job is to EAT and grow. The chrysalis's job is to TRANSFORM. The butterfly's job is to START THE CYCLE AGAIN with new eggs!"

**C. Introduce Key Vocabulary**
| Word | Child-Friendly Definition |
|------|---------------------------|
| Life Cycle | The way a living thing grows and changes |
| Metamorphosis | A big change—transforming into something new |
| Egg | The starting stage where life begins |
| Chrysalis | The resting stage where changes happen inside |

### Part 2: Guided Practice (15 minutes)

**A. Life Cycle Stations (Rotating)**
- **Station 1 - Sorting:** Match animal pictures to their life cycles
- **Station 2 - Sequencing:** Arrange 4-stage cards in order (butterfly)
- **Station 3 - Drawing:** Draw your own butterfly life cycle
- **Station 4 - Digital:** Explore the p5.js simulation

**B. Whole-Class Comparison**
- Compare butterfly to frog life cycle
- Create a Venn diagram on chart paper
- Highlight: Both start as eggs! Butterflies metamorphosis is complete; frogs lose their tail (gradual change).

### Part 3: Independent Practice (15 minutes)

**A. Ecosystem Connection Activity**
Students create a "Garden Connection Web":
1. Each student draws/colors a plant, animal, or insect
2. Connect them with lines showing:
   - What eats what
   - How they help each other
   - Where they live

3. **Focus Question:** "How does the butterfly help other living things in the garden?"

### Part 4: Closing (5 minutes)

**Exit Ticket:** "Draw and label one thing you learned about life cycles today."

**Preview:** "Tomorrow, we'll meet a scientist who studies life cycles for a living!"

## 7. Systems Literacy Connection

This lesson embraces Systems Literacy through:

### Big Ideas
1. **Interdependence:** Each life cycle stage depends on the previous one—nothing can be skipped
2. **Feedback Loops:** Butterflies pollinate plants → plants make seeds → new plants grow → more food for caterpillars
3. **Emergent Properties:** A caterpillar + time = butterfly (something completely new emerges!)
4. **Boundaries:** A garden ecosystem has living and non-living parts that interact

### Systems Thinking Language Used
- "What happens BEFORE this?"
- "What happens AFTER this?"
- "How is this CONNECTED to that?"
- "What would happen IF?"

### Visual: The Garden System

```
    ☀️ Sun Energy
       ↓
    🌱 Plants ←──🦋 Butterfly (pollinates)
       ↓           ↓
   Caterpillars ←──🐛 Eggs → 🦋 New Butterflies
       ↓
    🐦 Birds (eat caterpillars)
       ↓
    🌱 Soil enrichment → More plants!
```

## 8. First Peoples Integration

### Indigenous Perspectives on Life Cycles

**Honoring Traditional Knowledge:**

**Cedar Bark and the Salish Peoples (Coast Salish):**
> "The cedar tree is our relative. It gives us baskets, blankets, and longhouses. But we only take what we need. We thank the cedar for its gift. This is respect—taking care of our relationship with living things."

**Key Teaching Points:**
- Many First Peoples understand all living things as connected—plants, animals, humans, and the land
- Life cycles are part of the "Great Circle" of life
- Sustainable harvesting teaches respect for cycles
- Stories (oral tradition) pass down knowledge about seasons and life stages

### Classroom Integration

**Activity: "Our Gratitude Circle"**

**Connection to Place:**
- Acknowledge the traditional territory your school is on

## 10. Real-World Careers

### Career Spotlight: What Can You Be?

1. **Entomologist** (Bug Scientist)
   - Studies insects like butterflies
   - Works in labs, forests, universities
   - Helps farmers protect crops naturally

2. **Wildlife Biologist**
   - Studies how animals grow and change
   - Works to protect endangered species
   - Might work at zoos, nature reserves, or in the wild

3. **Botanist** (Plant Scientist)
   - Studies how plants grow from seeds
   - Works in gardens, farms, rainforests
   - Helps grow food and save endangered plants

4. **Farmer**
   - Understands life cycles of crops and animals
   - Feeds our communities
   - Works with nature, not against it

5. **Environmental Educator**
   - Teaches people about nature and cycles
   - Works at parks, museums, schools
   - Helps others care for our planet

### In-Class Connection:
**"Meet a Scientist"** - If possible, invite a local entomologist, Master Gardener, or Elder to visit (virtually or in-person) to share how they use life cycle knowledge in their work.

### Timing Guide

| Direct Instruction | 15 min | Use movements; keep energy high |
| Guided Practice | 15 min | Circulate; check understanding |

### Classroom Management Tips
- **Transitions:** Use a signal (chime, countdown, "Butterflies, freeze!")
- **Station Management:** Numbered table groups; rotate clockwise
- **Noise Level:** "Caterpillar whisper" for individual work, "Butterfly voice" for sharing
- **Scaffolding:** Have visual timers visible

### Common Misconceptions to Address
1. "Butterflies come from baby butterflies" → Clarify: they come from eggs, not other butterflies
2. "The chrysalis is a dead stage" → Explain: big changes happen inside!
3. "All animals change the same way" → Compare: humans don't metamorphose; frogs change gradually
4. "Life cycles are only about one animal" → Systems focus: everything is connected!

### Safety Notes
- If using live caterpillars (for observation): don't handle; observe from a distance
- Allergic concerns: check for latex/nut allergies before activities
- Online safety: p5.js runs locally; no internet required after loading

| IEP/504 | Pre-cut materials; extended time; verbal response option |

## 13. Parent Guide

### What We're Learning: Life Cycles

Dear Families,

This week in Science, your child is exploring **Life Cycles**—how all living things grow, change, and are connected!

### Key Concepts
- All living things go through stages: birth → growth → reproduction → death
- Butterflies go through complete metamorphosis: egg → caterpillar → chrysalis → butterfly
- Animals (like frogs) and plants (like pumpkins) also have life cycles
- All living things are connected in ecosystems

### Questions to Ask Your Child
1. "What are the 4 stages of a butterfly's life cycle?"
2. "How is a frog's life cycle different from a butterfly's?"
3. "What would happen if there were no more butterflies?"
4. "Can you show me with your hands the stages?"

### Try This at Home
- **Nature Walk:** Look for signs of life cycles in your yard or park (seeds, insects, birds' nests)
- **Cooking:** Make "Egg → Chick" cookies (round cookie + yellow coating = embryo → chick!)
- **Reading:** Visit the library for books about butterflies, frogs, or gardens
- **Garden:** Plant a seed and watch it grow—discuss the plant's life cycle

### Vocabulary to Review
- **Metamorphosis:** A big change in body form
- **Lifecycle:** The stages of an organism's life from beginning to end
- **Ecosystem:** Where living things interact with each other and their environment

### Questions? Concerns?
Please reach out! Science is all around us, and your child is a wonderful explorer.

## 14. p5.js Simulation Code

This interactive simulation lets students explore the butterfly life cycle and see how it's connected to a garden ecosystem.

### Instructions for Teachers
1. Go to [p5.js Web Editor](https://editor.p5js.org/)
2. Create a new sketch
3. Paste the code below
4. Press Play to run

### The Code

```javascript
/*
  Life Cycle Garden Simulation
  Grade 2 Science - Systems Literacy
  
  Click on different elements to learn about the life cycle!
*/

let stage = 0; // 0: egg, 1: caterpillar, 2: chrysalis, 3: butterfly
let butterflies = [];
let flowers = [];
let eggs = [];
let time = 0;
let showInfo = true;
let selectedAnimal = 'butterfly';

function setup() {
  createCanvas(800, 600);
  textAlign(CENTER, CENTER);
  
  // Create initial flowers
  for (let i = 0; i < 5; i++) {
    flowers.push(new Flower(random(100, 700), random(400, 550)));
  }
}

function draw() {
  // Sky gradient
  setGradient(0, 0, width, height, color(135, 206, 235), color(255, 255, 240));
  
  // Ground
  noStroke();
  fill(34, 139, 34);
  rect(0, 400, width, 200);
  
  // Sun
  fill(255, 215, 0);
  circle(700, 80, 80);
  
  // Draw flowers
  for (let f of flowers) {
    f.display();
  }
  
  // Draw and update butterflies
  for (let b of butterflies) {
    b.update();
    b.display();
  }
  
  // Draw eggs on leaf
  if (stage >= 0) {
    drawEggs();
  }
  
  // Draw life cycle stages based on current selection
  drawLifeCycleStages();
  
  // Information panel
  drawInfoPanel();
  
  // Instructions
  fill(50);
  textSize(16);
  text("Click LEFT/RIGHT arrows to change life cycle stage", width/2, 30);
}

function setGradient(x, y, w, h, c1, c2) {
  noFill();
  for (let i = y; i <= y + h; i++) {
    let inter = map(i, y, y + h, 0, 1);
    let c = lerpColor(c1, c2, inter);
    stroke(c);
    line(x, i, x + w, i);
  }
}

function drawLifeCycleStages() {
  let boxWidth = 120;
  let startX = 80;
  let y = 100;
  
  let labels = ["EGG", "CATERPILLAR", "CHRYSALIS", "BUTTERFLY"];
  let colors = [color(255, 255, 224), color(144, 238, 144), color(210, 180, 140), color(255, 182, 193)];
  
  for (let i = 0; i < 4; i++) {
    let x = startX + i * (boxWidth + 30);
    
    // Highlight current stage
    if (i === stage) {
      stroke(255, 0, 0);
      strokeWeight(3);
    } else {
      stroke(100);
      strokeWeight(1);
    }
    
    fill(colors[i]);
    rect(x, y, boxWidth, 80, 10);
    
    // Draw simple icon
    drawIcon(i, x + boxWidth/2, y + 35);
    
    // Label
    noStroke();
    fill(0);
    textSize(14);
    text(labels[i], x + boxWidth/2, y + 65);
  }
}

function drawIcon(iconType, x, y) {
  fill(50);
  noStroke();
  
  if (iconType === 0) { // Egg
    ellipse(x, y, 20, 25);
  } else if (iconType === 1) { // Caterpillar
    for (let i = 0; i < 5; i++) {
      circle(x - 15 + i * 8, y, 12);
    }
  } else if (iconType === 2) { // Chrysalis
    ellipse(x, y, 20, 35);
  } else if (iconType === 3) { // Butterfly
    // Wings
    fill(255, 100, 100);
    ellipse(x - 10, y - 5, 25, 20);
    ellipse(x + 10, y - 5, 25, 20);
    ellipse(x - 8, y + 10, 18, 15);
    ellipse(x + 8, y + 10, 18, 15);
    // Body
    fill(50);
    ellipse(x, y, 8, 25);
  }
}

function drawEggs() {
  fill(255, 255, 224);
  stroke(100);
  strokeWeight(1);
  
  // Draw leaf
  fill(34, 139, 34);
  beginShape();
  vertex(350, 380);
  vertex(450, 360);
  vertex(480, 400);
  vertex(420, 420);
  vertex(350, 400);
  endShape(CLOSE);
  
  // Draw eggs on leaf
  noStroke();
  fill(255, 255, 200);
  circle(380, 385, 15);
  circle(410, 390, 12);
  circle(395, 400, 10);
}

function drawInfoPanel() {
  fill(255, 255, 255, 230);
  stroke(100);
  rect(500, 180, 270, 180, 10);
  
  noStroke();
  fill(0);
  textAlign(LEFT, TOP);
  
  let infoText = "";
  let titleText = "";
  
  if (stage === 0) {
    titleText = "STAGE 1: THE EGG";
    infoText = "A butterfly starts as a tiny egg.\nThe mother butterfly lays eggs on a leaf.\nThis is the BEGINNING of the life cycle.\n\nThe egg is very small - like a dot!";
  } else if (stage === 1) {
    titleText = "STAGE 2: THE CATERPILLAR";
    infoText = "When the egg hatches, a caterpillar comes out!\n\nCaterpillars love to EAT leaves.\nThey grow bigger and bigger.\nThis is the GROWTH stage.";
  } else if (stage === 2) {
    titleText = "STAGE 3: THE CHRYSALIS";
    infoText = "The caterpillar attaches to a branch.\nIt makes a hard shell called a chrysalis.\n\nInside, amazing changes happen!\nIts body transforms into something new.";
  } else if (stage === 3) {
    titleText = "STAGE 4: THE BUTTERFLY";
    infoText = "A beautiful butterfly emerges!\nIt pumps blood into its wings to dry them.\n\nThen it flies, drinks nectar, and lays eggs.\nThe cycle STARTS AGAIN!";
  }
  
  textSize(16);
  fill(0, 100, 0);
  text(titleText, 515, 195);
  
  textSize(12);
  fill(50);
  text(infoText, 515, 225, 240, 150);
  
  textAlign(CENTER, CENTER);
}

function keyPressed() {
  if (keyCode === LEFT_ARROW) {
    stage = (stage - 1 + 4) % 4;
  } else if (keyCode === RIGHT_ARROW) {
    stage = (stage + 1) % 4;
  }
  
  // Spawn butterflies at stage 3
  if (stage === 3 && frameCount % 60 === 0) {
    butterflies.push(new Butterfly(random(100, 700), random(200, 350)));
  }
}

function mousePressed() {
  // Cycle through stages on click
  stage = (stage + 1) % 4;
  
  if (stage === 3 && butterflies.length < 5) {
    butterflies.push(new Butterfly(random(100, 700), random(200, 350)));
  }
}

// Flower class
class Flower {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.color = random([color(255, 105, 180), color(255, 255, 0), color(255, 165, 0), color(186, 85, 211)]);
  }
  
  display() {
    // Stem
    stroke(34, 139, 34);
    strokeWeight(3);
    line(this.x, this.y + 20, this.x, this.y + 60);
    
    // Petals
    noStroke();
    fill(this.color);
    for (let i = 0; i < 5; i++) {
      let angle = TWO_PI / 5 * i;
      circle(this.x + cos(angle) * 12, this.y + sin(angle) * 12, 15);
    }
    
    // Center
    fill(255, 215, 0);
    circle(this.x, this.y, 12);
  }
}

// Butterfly class
class Butterfly {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.vx = random(-2, 2);
    this.vy = random(-1, 1);
    this.wingAngle = 0;
  }
  
  update() {
    this.x += this.vx;
    this.y += this.vy;
    
    // Bounce off edges
    if (this.x < 0 || this.x > width) this.vx *= -1;
    if (this.y < 400 || this.y > 550) this.vy *= -1;
    
    // Random direction changes
    if (random() < 0.02) {
      this.vx = random(-2, 2);
      this.vy = random(-1, 1);
    }
    
    // Wing flutter
    this.wingAngle += 0.3;
  }
  
  display() {
    push();
    translate(this.x, this.y);
    
    let wingFlap = sin(this.wingAngle) * 30;
    
    // Wings
    fill(255, 182, 193);
    stroke(200, 100, 100);
    strokeWeight(1);
    
    // Left wings
    push();
    rotate(radians(-20 + wingFlap));
    ellipse(-15, 0, 30, 20);
    ellipse(-10, 12, 20, 15);
    pop();
    
    // Right wings
    push();
    rotate(radians(20 - wingFlap));
    ellipse(15, 0, 30, 20);
    ellipse(10, 12, 20, 15);
    pop();
    
    // Body
    fill(50);
    noStroke();
    ellipse(0, 0, 6, 25);
    
    // Antennae
    stroke(50);
    strokeWeight(1);
    noFill();
    arc(-3, -10, 10, 10, PI + 0.5, TWO_PI - 0.5);
    arc(3, -10, 10, 10, PI + 0.5, TWO_PI - 0.5);
    
    pop();
  }
}
```
